🎓 Student Management System (PHP Session Based)
📌 Project Overview

This is a simple and professional Student Management System developed using PHP and Sessions.
It allows users to:

➕ Add new students

📖 View student records

✏️ Edit student details

🗑️ Delete student records

The project is designed with a clean university-style admin panel theme using professional color combinations and responsive layout.

🛠 Technologies Used

PHP

HTML5

CSS3

PHP Sessions (for temporary data storage)

✨ Features

Professional admin dashboard UI

Inline editing functionality

Confirmation before deletion

Responsive design

Clean corporate color theme

Separate Registration Form page

Organized and user-friendly layout

📂 Project Structure
/project-folder
│── form.php        (Student Registration Form)
│── students.php    (Students List + Edit/Delete)
│── README.md
🚀 How to Run the Project

Install XAMPP / Laragon / WAMP

Place the project folder inside the htdocs or www directory
Start Apache server

Open browser and visit:
http://localhost/srudents/form.php
